# CleanPixels

**CleanPixels** is a software I developed for personal use to make my daily
workflow easier, as I always compress and remove EXIF metadata from images
before uploading them to the internet. Since this was built primarily for my
own needs, it hasn't been optimized for large-scale use or extensively tested
across different system configurations. Its public release is merely for
educational and documentary purposes; the codebase is very simple, making it
highly interesting for beginners to study.

However, because it is developed in Python using cross-platform libraries, it
will most likely work perfectly on your computer. It can be a great tool for
anyone looking to replace web-based alternative compressors.

By running locally, CleanPixels brings all the security, privacy, and speed
advantages of using open-source, local software.
